# Fluidd - my fork

## Changes

- add webrtc-go2rtc support
- ui chaneges for centering tool controls

Fluidd is a free and open-source Klipper web interface for managing your 3d printer.

![Fluidd](/docs/assets/images/preview_sliced.png "Fluidd")

## Docker

We have an [official docker image](https://github.com/vajonam/fluidd/pkgs/container/fluidd). This is updated for each release and on each commit.
